# 台灣收納盒尺寸搜尋 Demo

用尺寸反查台灣買得到的收納盒、收納箱、抽屜盒與整理盒。

## Demo 功能

- 輸入可用空間的寬、深、高
- 以商品外部尺寸判斷是否放得下
- 顯示商品圖、名稱、品牌、尺寸、價格、通路、購買連結
- 支援通路、有蓋、可堆疊篩選
- 商品資料先用 `products.js` 人工整理

## 搜尋規則

商品符合條件的定義：

```txt
商品外寬 <= 空間寬
商品外深 <= 空間深
商品外高 <= 空間高
```

尺寸統一使用：

```txt
寬 × 深 × 高，單位 cm
```

## 檔案

```txt
index.html    頁面結構
styles.css    視覺樣式
products.js   商品示範資料
app.js        搜尋與篩選邏輯
```
